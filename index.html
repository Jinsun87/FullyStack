Using Images

  1. Using Images from other website

    2. Code:

    <img src="https://soccersouls.com/wp-content/uploads/2018/02/Cristiano-Ronaldo-real-madrid-1024x683.jpg" alt="CR7 on fire">

    Explanation

    Open the image tag and copy paste the image address from any website inside the source attribute. (alt) attribute is also used to display a text when the image cannot be shown for any reasons.
  • Using images from local or hosted webserver

    • Code

    <img src="images/messi.jpg" alt="CR7 on fire">

    Explanation:

    to use the image from local or hosted server, the image needs to there in the same directory as the html file. (src="example.jpg")or (src="folder/example.jpg") if the image is inside any folder within the same directory.
